# 검침 청구 단체문자 웹 MVP

한셀/Excel/CSV 시트를 불러와 수신자별 청구 상세 카드를 만들고, 문자 앱으로 발송 큐를 여는 정적 웹앱입니다.

## 사용 방법

1. `index.html`을 브라우저에서 엽니다.
2. `.csv`, `.xls`, `.xlsx` 파일을 선택하거나 끌어놓습니다.
3. 단체문자 문구를 작성합니다.
4. `{상세링크}` 변수를 문자에 포함합니다.
5. `발송 큐 만들기`를 눌러 수신자별 문자 앱 열기 순서를 확인합니다.

## 시트 헤더

필수 발송용 헤더:

- 전화번호

카드 표시용 헤더:

- 소속
- 성명
- 거주지명
- 호수
- 전월검침
- 당월검침
- 당월사용량
- 국고지원량
- 초과사용량
- 청구금액
- 국고지원금
- 초과금액
- 사용인원
- 인원별부담금액

## 참고

웹 브라우저는 보안상 SMS를 자동 발송할 수 없습니다. 이 앱은 문자 내용을 만들고 휴대폰 문자 앱을 여는 방식으로 테스트합니다.
